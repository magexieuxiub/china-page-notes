# china-page-notes

本仓库用于归档和发布多个独立的 HTML 页面。页面内容覆盖不同主题，不针对任何单一域名或具体网站，仅作为静态资料的整理与展示。

## 项目简介

本项目是一个通用型 HTML 页面归档仓库。所有页面均为独立文件，不依赖外部数据库或后端服务。仓库主要目的为：

- 归档各类独立页面，便于长期保存与访问
- 通过 GitHub Pages 等静态托管服务发布页面内容
- 提供清晰的目录结构，方便浏览与检索

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库主页（可选，用于导航）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件为一个独立页面，文件名通常反映页面主题。
- `assets/` 目录按需组织，可包含子目录（如 `images/`、`css/`、`js/`）。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，尽量将样式与脚本内嵌或引用自 `assets/`。
- 页面内容可能随时间更新或补充，但不会删除已有页面（除非存在明显错误）。
- 归档页面不保证与任何外部网站或服务保持同步，仅反映归档时的状态。

## 维护说明

- 本仓库由个人维护，更新频率不固定。
- 欢迎通过 Issue 提交页面归档建议或报告问题。
- 不接受直接修改仓库内容的 Pull Request，除非属于明显修复（如链接失效、样式错乱等）。
